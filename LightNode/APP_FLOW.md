# APP_FLOW.md

LightNode 用户流程与页面导航规范

版本：v0.1
日期：2026-03-16

---

# 1. 文档目的

本文档定义 **LightNode v0.1 的页面结构、页面跳转规则、用户操作步骤、成功状态、错误状态和空状态**。

目标：

* 明确用户在应用中的每一个导航路径
* 防止 AI 或开发过程中产生未定义的页面或流程
* 为前端和后端开发提供统一行为规范

**约束规则：**

1. 只有本文档中定义的页面才允许存在
2. 未定义流程不得由 AI 自动创建
3. 主页面仅有两个：

   * 项目主页
   * 项目导图页
4. 其他界面优先使用：

   * 弹窗
   * 侧边栏
   * 抽屉面板

---

# 2. 页面总表

LightNode v0.1 页面结构如下：

| 页面名称       | 类型   | 说明      |
| ---------- | ---- | ------- |
| 启动页 / 项目主页 | 主页面  | 项目入口    |
| 新建项目弹窗     | 弹窗   | 创建项目    |
| 导入弹窗       | 弹窗   | 导入内容    |
| 解析状态页      | 页面   | AI解析过程  |
| 项目导图页      | 主页面  | 核心工作区   |

| 搜索面板       | 弹出面板 | 节点搜索    |
| 待处理侧边栏     | 侧边栏  | 导入失败或草稿 |
| 动态重构预览弹窗   | 弹窗   | 结构更新确认  |
| 设置页        | 页面   | 偏好设置    |
| 登录/注册弹窗    | 弹窗   | 账户系统    |
| 订阅弹窗       | 弹窗   | 付费入口    |

# 3. 全局导航规则

## 3.1 应用启动

应用启动后：

```
启动应用
→ 加载本地数据库
→ 进入 项目主页
```

如果数据库读取失败：

```
显示错误页
按钮：重新加载
```

---

## 3.2 主导航结构

主导航只包含：

```
项目主页
项目导图页
```

其他界面全部通过：

* 弹窗
* 侧边栏
* 抽屉

打开。

---

## 3.3 页面跳转总览

| 当前页面   | 操作     | 结果     |
| ------ | ------ | ------ |
| 项目主页   | 点击项目   | 项目导图页  |
| 项目主页   | 点击新建项目 | 新建项目弹窗 |
| 新建项目成功 | 自动     | 导入弹窗   |
| 导入提交   | 自动     | 解析状态页  |
| 解析成功   | 自动     | 项目导图页  |
| 导入失败   | 自动     | 待处理    |
| 项目导图页  | 点击搜索   | 搜索面板   |
| 项目导图页  | 点击节点   | 节点详情   |
| 项目导图页  | 导入新内容  | 动态重构弹窗 |

---

# 4. 核心流程

本应用共有 **6 条核心用户流程**：

1. 创建第一个项目
2. 导入内容并生成导图
3. 查看与复习知识
4. 搜索节点
5. 导入新内容并重构结构
6. 导入失败后的待处理流程

---

# 5. 页面流程定义

---

# 页面 1：项目主页

## 页面目标

展示用户已有项目并提供入口。

---

## 页面元素

* 新建项目按钮
* 项目卡片列表
* 搜索入口
* 待处理入口
* 设置入口
* 账户入口
* 订阅入口

---

## 用户操作

### 创建项目

```
点击 新建项目
→ 打开 新建项目弹窗
```

---

### 打开项目

```
点击 项目卡片
→ 打开 项目导图页
```

---

## 成功状态

正常显示项目卡片列表。

卡片样式：

* 圆角矩形
* 粗边框
* 左侧几何图标
* 右侧2-6字描述

---

## 空状态

```
还没有知识地图
先导入一段 AI 对话吧
```

按钮：

```
新建项目
```

---

## 错误状态

数据库读取失败：

```
无法加载项目
[重新加载]
```

---

# 页面 2：新建项目弹窗

## 用户步骤

```
点击 新建项目
→ 弹出 新建项目弹窗
```

输入：

* 项目名称

选择：

* 使用默认主文件夹
* 创建主文件夹

点击：

```
创建并导入
```

---

## 成功状态

```
关闭弹窗
→ 自动打开 导入弹窗
```

---

## 错误状态

项目名为空：

```
请输入1-24字项目名
```

名称过长：

```
项目名不能超过24字
```

---

# 页面 3：导入弹窗

导入方式：

| 方式   | 说明        |
| ---- | --------- |
| 分享链接 | AI平台分享链接  |
| 文件导入 | JSON / MD |
| 粘贴文本 | AI对话      |

---

## 链接导入流程

```
粘贴链接
→ 点击继续
→ 授权确认
→ 点击开始解析
```

成功：

```
关闭弹窗
→ 解析状态页
```

---

## 文件导入流程

```
选择文件
→ 点击解析
→ 解析状态页
```

---

## 粘贴文本流程

```
粘贴对话
→ 点击解析
→ 解析状态页
```

文本预处理：

* 移除时间戳
* 移除角色标记
* 保留代码块
* 保留公式

---

## 错误状态

链接错误：

```
无法识别该链接
```

解析失败：

```
该内容暂不支持解析
```

用户可选择：

```
存入待处理
```

---

# 页面 4：解析状态页

显示解析进度：

```
上传内容
解析文本结构
生成摘要
构建节点树
生成导图
```

---

## 成功

解析完成：

```
自动进入 项目导图页
```

首次进入展示动画：

```
模块淡出
标题归位
视野拉远
导图展开
```

---

## 失败流程

系统自动：

```
自动重试一次
```

仍失败：

显示：

```
解析失败
[手动重试]
[保存到待处理]
```

---

## 降级策略

如果导图生成失败：

```
降级为大纲式极简笔记
```

---

# 页面 5：项目导图页（核心页面）

主要区域：

```
顶部工具栏
导图画布
节点详情面板
```

---

## 工具栏

按钮：

* 返回主页
* 搜索
* 导入内容
* 新增节点
* 撤销
* 重做
* 设置

---

## 节点交互规则

### 第一次点击节点

显示：

```
极简概述卡片
```

内容：

* 节点标题
* 极简摘要

离线可用。

---

### 第二次点击节点

打开：

```
节点详情面板
```

展示：

* 详细论述
* 关键定义
* 流程说明
* 代码块
* Mermaid
* 公式

---

### 低置信度节点

显示：

```
黄色问号
```

---

## 导图操作

支持：

* 拖拽
* 缩放
* 平移

深度超过5层：

```
视觉弱化
```

---

# 页面 6：节点详情面板

内容：

* 标题
* 极简概述
* 详细论述
* 定义
* 推导逻辑
* 原始片段引用

按钮：

```
详尽一点
简短一点
编辑
```

---

## 编辑操作

支持：

* 编辑内容
* 重命名
* 新增节点
* 调整层级

---

## 成功状态

```
已保存
```

用户修改后：

```
节点标记为 用户编辑
```

AI 不再自动修改。

---

# 页面 7：搜索面板

打开方式：

```
点击搜索
```

流程：

```
输入关键词
→ 显示匹配节点
→ 点击结果
→ 导图定位节点
```

---

## 聚焦模式

用户可：

```
只查看该分支
```

---

## 错误状态

无结果：

```
没有找到相关节点
```

---

# 页面 8：待处理侧边栏

来源：

* 导入失败
* 用户主动保存

---

显示：

```
草稿列表
```

每项显示：

* 来源类型
* 导入时间

操作：

```
重新解析
删除
```

---

# 页面 9：动态重构预览

触发条件：

```
已有项目再次导入内容
```

流程：

```
解析完成
→ 检测结构变化
→ 弹出重构预览
```

---

显示：

```
旧节点变灰
新节点高亮
结构变化路径
```

操作：

```
Accept
稍后再说
取消
```

---

## Accept

```
导图平滑过渡
新增分支高亮
```

---

选项：

# 6. 离线规则

离线可用：

* 节点标题
* 极简概述
* Mermaid渲染
* 公式渲染

在线必需：

* 导入
* AI解析
* 第二击详细内容
* 动态重构

---

# 7. 撤销规则

可撤销操作：

* 删除节点
* 移动分支
* 重命名
* 编辑内容
* 接受 AI 重构

---

# 8. 降级策略

解析失败：

```
自动重试一次
```

导图失败：

```
降级为大纲模式
```

---

# 9. 性能目标

解析时间：

```
20-30 秒
```

最大解析：

```
100k token
```

---

# 10. 不允许的行为

AI 不得：

* 新增未定义页面
* 新增未定义流程
* 自动改变导航结构
* 自动增加功能
